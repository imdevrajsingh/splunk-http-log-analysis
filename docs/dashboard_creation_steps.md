# Steps to Create Dashboards in Splunk

1. Go to Splunk Home → Dashboards → Create New Dashboard
2. Name the dashboard (example: Sensitive File Access Attempts)
3. Choose "Classic Dashboard"
4. Click "Create Dashboard"
5. Click "Add Panel"
6. Select "New Search"
7. Paste the SPL query from /spl/queries.md
8. Choose visualization (Table, Pie Chart, Bar Chart, etc.)
9. Save Panel
10. Repeat for all panels / dashboards
